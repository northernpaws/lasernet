Raspberry Pi 1 and Zero W
=========================

Boot Media
----------
Use the image ``platform-rpi1/images/hd.img`` and write it onto your SD Card.

Serial Console
--------------
The serial boot console is available at J8 (the GPIO header) with 115200
baud, 8N1 on the following pins:

* J8.6: GND
* J8.8: Tx
* J8.10: Rx
